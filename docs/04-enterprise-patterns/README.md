# Enterprise Patterns

This directory is reserved for patterns such as DTO, Repository, Specification, Domain Event, Outbox, Inbox, Retry, Circuit Breaker, and similar application-scale concerns.

The repository covers selected enterprise patterns when they clarify real application boundaries. Add new standalone guides only when they stay problem-first and do not turn enterprise patterns into default architecture.

Future additions should explain the boundary pressure, simpler alternative, and failure modes that justify the pattern.
