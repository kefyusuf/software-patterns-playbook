# Examples

This directory contains small, readable examples that support the documentation.

The repository does not aim to ship full runnable framework applications. Examples should stay minimal, focused, and tied to specific pattern or scenario guides.

Current coverage:

- [Go examples](./go/README.md) for runnable pattern and scenario slices

Still intentionally conceptual:

- architecture guides
- anti-pattern guides
- introduction and checklist documents

Selection rule:

- add an example only when it clarifies a boundary better than prose alone
